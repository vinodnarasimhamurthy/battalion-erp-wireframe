Administration / System Architecture

System Architecture

Fully self-contained, air-gapped deployment on the unit LAN
INDRA runs entirely on the unit's own closed network. There is no connection of any kind to the internet, cloud, email, the Army Data Network or PAO/CDA systems. Outputs are produced locally and handled by hand; all data, identity and backups reside on the unit LAN.

Isolation

Absolute air gap — no internet, cloud, email, ADN or PAO/CDA link. Nothing crosses the perimeter in either direction.

Central core

One hardened server room runs app, PostgreSQL (RLS + audit), identity and analytics; backups stay on-unit (NAS + offline safe).

Distributed & scoped

Ten terminals across the lines on one unit-owned LAN; each Coy Cdr terminal is row-level scoped to its own company.
